# SD Bintang Ceria

Landing page untuk SD Bintang Ceria — sekolah dasar unggulan yang menyenangkan, kreatif, dan berprestasi.

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS 4 + shadcn UI
- **Animasi**: Motion (Framer Motion)
- **Backend**: Convex
- **Auth**: Hercules Auth

## Fitur

- Hero section dengan animasi floating emoji
- Section keunggulan sekolah
- Program belajar per jenjang kelas
- Galeri foto sekolah
- Testimoni orang tua murid
- Form kontak/pendaftaran
- Responsive design (mobile-friendly)
- Tema warna orange yang ceria

## Menjalankan Lokal

```bash
pnpm install
pnpm dev
```
